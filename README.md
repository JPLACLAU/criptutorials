## Instalation and local edit

npm config set legacy-peer-deps true  
npm install --legacy-peer-deps  
npm run build --legacy-peer-deps  
npm start

## deploy to GCP GAE

Add .npmrc file in the root with legacy-peer-deps=true in the file

gcloud init  
gcloud app deploy
